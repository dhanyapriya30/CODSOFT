## 📖 Project Description

The **Currency Converter** is a Java console application developed as **Task 2** of the **CodeSoft Java Programming Internship**. The application enables users to convert currencies using predefined exchange rates through a simple menu-driven interface.

The program supports conversions between **US Dollar (USD)**, **Indian Rupee (INR)**, and **Euro (EUR)**, providing quick and accurate currency calculations.

---

## ✨ Features

* Convert **USD to INR**
* Convert **INR to USD**
* Convert **EUR to INR**
* Convert **INR to EUR**
* Menu-driven console application
* Simple and user-friendly interface
* Instant currency conversion results

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/CurrencyConverter.git
```

2. Navigate to the project folder.

```bash
cd CurrencyConverter
```

3. Compile the Java program.

```bash
javac currencyconverter.java
```

4. Run the application.

```bash
java currencyconverter
```

## 🔄 Supported Currency Conversions

| Conversion |         Exchange Rate |
| ---------- | --------------------: |
| USD → INR  |     1 USD = 83.47 INR |
| INR → USD  | 1 INR = 0.0119804 USD |
| EUR → INR  |     1 EUR = 89.10 INR |
| INR → EUR  |    1 INR = 0.0113 EUR |

> **Note:** The exchange rates used in this project are fixed for educational purposes and may differ from real-time market exchange rates.
